# **App Name**: Silent Sentinel

## Core Features:

- Real-time Noise Detection: Capture ambient noise levels using the device microphone.
- AI Noise Classification: Use a pre-trained AI model tool to classify the type of detected noise (e.g., traffic, construction, human speech).
- Threshold Monitoring: Monitor noise levels against user-defined thresholds and trigger alerts when exceeded.
- Geolocation Tagging: Tag noise recordings with geolocation data to map noise pollution hotspots.
- Interactive Heatmap: Display noise pollution levels on an interactive heatmap.

## Style Guidelines:

- Primary color: Soft blue (#79B4B7) to represent calmness and monitoring.
- Background color: Light gray (#E8F0F2) for a clean, unobtrusive backdrop.
- Accent color: Muted orange (#D77A61) to draw attention to alerts and actionable items. 
- Body and headline font: 'PT Sans', a modern sans-serif that offers both approachability and professional style.
- Use clean, minimalist icons to represent noise types and monitoring functions.
- Dashboard-style layout, optimized for quick review of noise levels, threshold breaches and geolocation heatmap.
